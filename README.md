# Shayan Sheikh Portfolio Website

## Project Overview
This is a responsive personal portfolio website for Shayan Sheikh, a freelance graphic artist, web designer, and illustrator. The website showcases professional work samples, information about the artist, and contact details in a modern, clean design.

## Features
- **Responsive Design**: Built with Bootstrap 4.6 for full responsiveness across all devices
- **Modern UI**: Clean and professional interface with a consistent color scheme
- **Portfolio Showcase**: Gallery section displaying artistic works and projects
- **About Section**: Professional summary and download option for resume/portfolio
- **Contact Form**: Interactive form for potential clients to reach out
- **Social Media Integration**: Links to professional social media profiles

## Technologies Used
- **HTML5**: Semantic markup for content structure
- **CSS3**: Custom styling with responsive design principles
- **Bootstrap 4.6**: Framework for responsive grid layout and components
- **jQuery**: JavaScript library for interactive elements
- **Font Awesome 4.7**: Icon library for social media and UI elements

## Sections
1. **Navigation Bar**: Responsive menu with links to all sections
2. **Header**: Introduction with avatar and professional title
3. **Portfolio**: Gallery of work samples in a responsive grid
4. **About**: Professional summary and background information
5. **Contact**: Form for potential clients to send inquiries
6. **Footer**: Location information, social media links, and copyright notice

## Design Elements
- **Color Scheme**: Primary teal (#1bbc9b) with dark accents (#2D3E50, #1a242f)
- **Typography**: Clean, professional font selection via Bootstrap
- **Visual Hierarchy**: Clear section organization with consistent headings
- **Whitespace Management**: Balanced spacing for optimal readability

## Responsive Behavior
- **Desktop**: Full layout with multi-column content
- **Tablet**: Adjusted spacing and sizing for medium screens
- **Mobile**: Single column layout with optimized navigation

## Future Enhancements

## Installation and Usage
1. Clone or download the repository
2. Open index.html in any modern web browser
3. No additional setup required as the site uses CDN-hosted dependencies
